Observations about RecycledViewPool

A static nested class of RecyclerView, providing mechanism to share ViewHolder between multiple RecyclerViews.

Fields

•

mScrap

a SparseArray, key of which is ViewType defined in Adapter, while value is a List of ViewHolders

•

mMaxScrap

a SparseIntArray, key of which is ViewType defined in Adapter, while value is the max count of ViewHolders to cache for this specific ViewType
